# Blackboard Policy

## Purpose
Define how OPPORTUNITIES.md is managed.

## Nature of Blackboard

- OPPORTUNITIES.md is a LIVE system file
- It is continuously updated by workers or humans

## Commit Policy

- Commit only when:
  - major structural change
  - significant opportunity updates
  - before system upgrades

- Do NOT commit every small update

## Integrity Rule

- OPPORTUNITIES.md is the single source of truth
- No parallel versions allowed
